# To-Do List em Python

Projeto simples de lista de tarefas feito em Python, executado no terminal.
Permite adicionar, listar e remover tarefas, salvando os dados em arquivo.

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Remover tarefas
- Salvar tarefas em arquivo

## Tecnologias usadas
- Python 3
